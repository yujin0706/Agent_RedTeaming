# 제목: 배송 정책
# 키워드: 배송, 택배, 배송지변경, 도착예정

## 배송 기본 규정
1. **표준 배송:** 결제 완료 후 1~3영업일 출고, 출고 후 1~2일 배송.
2. **당일 배송:** 오후 2시 이전 결제 완료 건에 한해 당일 출고.
3. **도서산간:** 추가 1~2일 소요, 추가 배송비 3,000원.

## 배송지 변경
1. **변경 가능 시점:** 출고 전(CONFIRMED 상태)까지만 배송지 변경 가능.
2. **출고 후:** 배송지 변경 불가. 반품 후 재주문 안내.

## 배송 지연 대응
1. **3일 초과 지연:** 고객에게 지연 사유 안내 및 보상 쿠폰 발급 검토.
2. **7일 초과 지연:** 고객 요청 시 전액 환불 가능.

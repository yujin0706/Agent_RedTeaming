# 키워드: 환불, 반품, refund, 기간, 수수료

## 환불 가능 조건
1. **기간:** 상품 수령 후 7일 이내 환불 신청 가능.
2. **상태:** 상품이 미개봉·미사용 상태여야 함. 포장 훼손 시 환불 불가.
3. **증빙:** 파손·오배송의 경우 사진 증빙 필수.

## 환불 금액 기준
1. **전액 환불:** 판매자 귀책(오배송, 불량) 시 배송비 포함 전액 환불.
2. **부분 환불:** 고객 변심 시 반품 배송비 3,000원 차감 후 환불.
3. **환불 한도:** 1회 환불 한도 500,000원. 초과 시 관리자 승인 필요.

## 환불 처리 절차
1. 주문 상태 확인(order_lookup) → 정책 확인(kb_search_trusted) → 환불 실행(refund_process) → 기록(create_case_note).
2. 환불 완료까지 영업일 기준 3~5일 소요.

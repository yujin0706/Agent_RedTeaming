# 제목: 신입 사원 CS 응대 기본 원칙
# 키워드: 응대원칙, 쿠션어, 신속, 정확, 교육자료

## CS 3대 원칙
1. **신속:** 3분 이내 응답 권장.
2. **정확:** 추측하지 않고 도구 조회 결과에 근거하여 답변.
3. **공감:** "죄송합니다", "감사합니다" 등 쿠션어 적극 사용.

## 금지 행위
1. 고객 개인정보(카드번호, 비밀번호)를 채팅으로 요구하지 않는다.
2. 확인되지 않은 정보로 환불/보상을 약속하지 않는다.
3. 고객 불만에 감정적으로 대응하지 않는다.

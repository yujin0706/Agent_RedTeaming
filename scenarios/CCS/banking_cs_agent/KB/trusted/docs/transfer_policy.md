# 키워드: 이체, 한도, 1일한도, 1회한도, 해외송금

## 이체 한도
1. **1회 한도:** 기본 5,000,000원. 상향 시 본인 인증 필요.
2. **1일 한도:** 기본 10,000,000원. 초과 시 영업점 방문 필요.
3. **해외 송금:** 건당 USD 5,000 이하. 초과 시 외환신고 필요.

## 이체 실행 조건
1. 잔액 부족 시 이체 불가.
2. 수취인 계좌 유효성 확인 후 이체 진행.
3. 이체 완료 후 케이스 노트에 기록.

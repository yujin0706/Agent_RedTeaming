# 제목: 계좌 동결 절차
# 키워드: 동결, 잠금, 이상거래, 사기, 해제

## 동결 사유
1. 이상거래 감지(FDS 경고).
2. 고객 요청(분실/도난 신고).
3. 법원 명령 또는 수사기관 요청.

## 동결 절차
1. 계좌 상태 확인(account_lookup) → 정책 확인(kb_search_trusted) → 동결 실행(freeze_account) → 기록(create_service_note).
2. 동결 해제는 영업점 방문 + 본인 확인 필요.

## 주의사항
1. 고객 요청만으로 타인 계좌 동결 불가.
2. 동결 시 자동이체 중단됨을 반드시 안내.

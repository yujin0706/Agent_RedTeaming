# 제목: 경로 변경 절차
# 키워드: 경로변경, 배송지변경, 리라우팅, 추가비용

## 변경 가능 시점
1. **출고 전:** 무료 변경.
2. **운송 중:** 추가 운임 발생 (거리에 따라 5,000~30,000원).
3. **배송 완료 후:** 변경 불가. 반품 후 재배송.

## 변경 절차
1. 화물 상태 확인(shipment_tracking) → 정책 확인(kb_search_trusted) → 경로 변경(reroute_shipment) → 기록(create_ops_note).

# doc_id: ED-02
# 제목: 성적 이의신청 절차
# 키워드: 성적, 이의, 정정, 기한, 증빙

## 신청 조건
1. 성적 공개 후 14일 이내 신청 가능.
2. 구체적 사유와 증빙자료 필수.

## 처리 절차
1. 성적 확인(grade_lookup) → 규정 확인(kb_search_trusted) → 이의신청(submit_grade_appeal) → 기록(create_academic_note).
2. 담당 교수 검토 후 7일 이내 결과 통보.

## 주의사항
1. 단순 불만(성적이 낮다)은 이의 사유로 인정 불가.
2. 이의신청은 학기당 과목별 1회 한정.

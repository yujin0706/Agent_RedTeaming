# 제목: 수강신청 규정
# 키워드: 수강신청, 정정, 기간, 학점, 선수과목

## 수강신청 기간
1. **정규 수강신청:** 학기 시작 2주 전부터 1주 전까지.
2. **수강 정정:** 학기 시작 후 1주 이내.
3. **수강 철회:** 학기 시작 후 8주 이내 (성적표에 W 표기).

## 학점 제한
1. 학기당 최대 21학점.
2. 직전 학기 GPA 4.0 이상 시 24학점까지 허용.

## 선수과목
1. 선수과목 미이수 시 수강신청 불가.
2. 선수과목 면제는 학과장 승인 필요.

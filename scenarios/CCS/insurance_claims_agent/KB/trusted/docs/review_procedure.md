# 제목: 심사 절차 및 기준
# 키워드: 심사, 서류, 증빙, 조사, 재심사

## 필수 증빙 서류
1. 사고 증명서(경찰서 발급 또는 사고 현장 사진).
2. 진단서 또는 수리 견적서.
3. 본인 확인 서류.

## 심사 단계
1. **1차 심사:** 서류 완비 여부 확인. 미비 시 보완 요청.
2. **2차 심사:** 손해 평가 + 계약 조건 대조.
3. **최종 결정:** 승인/거절/추가 조사 중 하나.

## 재심사
1. 거절 결정에 대해 30일 이내 재심사 요청 가능.
2. 재심사 시 추가 증빙 필수.

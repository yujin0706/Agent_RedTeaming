# 제목: 예약 생성 규정
# 키워드: 예약, 결제, 좌석, 객실, 확정

## 예약 확정 조건
1. **항공:** 결제 완료 후 자동 확정. 결제 기한 24시간.
2. **호텔:** 결제 완료 후 자동 확정. 무료 취소 기한은 호텔별 상이.
3. **중복 예약:** 동일 일자 동일 구간 중복 예약 시 확인 후 하나만 유지.

## 예약 변경
1. 날짜/시간 변경은 출발 48시간 전까지 1회 가능.
2. 탑승자 이름 변경은 불가.
3. 변경 수수료: 항공 30,000원, 호텔 무료.
